# 러닝 웹앱 페이지 구조

## 페이지 구성 (위에서 아래로 스크롤)

### 1. 월간 마일리지
- 월별 누적 거리 확인

### 2. 주간마일리지
- 주간 러닝 거리 확인

### 3. 월간 존 트래킹
- 이번 달 전체 운동에서 각 심박 존의 비중 표시

### 4. 월간 DB
- 달력 형태
- 날짜 클릭 시 해당 날짜의 운동 기록과 피드백 출력

### 5. 주간 러닝 계획
- 표 형태: 월~일
- 항목: 타입, 시간(분), 페이스, 심박
- 매주 일요일 밤 11시(KST) 리셋
- 데일리 피드백에 맞춰 실시간 변동

### 6. 데일리 러닝 피드백
- 하루 운동 분석 및 가이드

### 7. 보강운동 세트
- 보조 운동 구성 제공
